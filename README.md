# 向前一步

一个本地优先、可安装到手机桌面的 PWA。它奖励尊重、自然的主动交流行为，而不是把联系方式或对方的回应当作成绩。

## 当前功能

- 0–5 级渐进式训练阶梯
- 默认每周 4 个完整训练日、每个训练日完成 5 次实际接近；白天偶遇和晚间练习都计入
- 绿色表示完成 5 次，蓝色表示已行动 1–4 次，黄色表示因现实原因顺延，灰色表示休息或未安排
- 下雨、临时有事、状态不佳、场地不合适和其他顺延原因记录与周统计
- 间隔训练后的主动休息券，不使用断签清零
- 行动、礼貌退出、判断不适合、回避四种记录
- 回避想法的本机语义归类与每周“如果—那么”计划
- 根据焦虑难度生成不同纹样和稀有度的勇气卡
- 勇气值、称号、成就与象征性点亮地图
- 真实奖赏契约与吃饭以外的实际开销账本
- 全部数据仅保存在浏览器本机，可导出 JSON
- Service Worker 离线缓存和 PWA 桌面安装配置

## 本地运行

PWA 的离线缓存与安装需要通过 `http://` 或 `https://` 打开，不能直接双击 `index.html`。

如果电脑上有 Python：

```powershell
python -m http.server 4173
```

然后访问 `http://localhost:4173`。

要在华为手机上长期使用，需要把该目录部署到 HTTPS 地址。用华为浏览器打开后，点击右下角 `∷`，选择“添加至”→“桌面”；HarmonyOS 5 可直接选择“添加至桌面”。桌面图标如果带蓝色小闪电，表示华为浏览器正在以 PWA/快应用方式创建入口，属于正常现象。

## 数据与边界

- 不启用 GPS，不记录遇到他人的真实位置。
- 不上传数据，不保存他人照片、姓名或可识别特征。
- 拒绝、冷淡或回避出现时，训练流程要求立即结束。
- 每日达到练习上限后仍可自愿记录，但不再增加勇气值，避免强迫性搜寻目标。
